# freebee-landing

Landing page built with Next.js 16 and React 19.

## Local development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import the GitHub repository in Vercel.
3. Keep the default Next.js framework settings.
4. Deploy.
